# 08. Dark Rodo: backpacks de runas

## Que es

Extension del NPC de runas para vender backpacks completas de las runas mas usadas, en vez de obligar a comprar todo de a una.

## Por que valio la pena

- acelera mucho el resupply de mages
- baja spam de chat con cantidades
- hace mas comodo preparar hunts o PvP

## Que conviene conservar al portarlo

- backpacks listas con cantidad cerrada
- precio = contenido + backpack
- aliases cortos y largos
- integracion con confirmacion del NPC

## Riesgos

- si las charges del contenido se crean mal, el pack llega defectuoso
- si cambian precios base de runas, el pack queda desalineado
- si el helper de backpacks es demasiado generico y mal validado, puede usarse mal en otros NPCs

## Portabilidad

Alta. La idea es simple y de mucho valor practico.
