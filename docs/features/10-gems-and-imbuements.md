# 10. Gemas e imbuements

## Que es

Sistema de progresion lateral basado en:

- gemas chicas que caen de monstruos
- fusion de `20 -> 1` gema grande
- salida economica via NPC comprador
- uso de gemas grandes para imbuir items equipados

## Por que valio la pena

- agrega objetivos de farmeo
- crea decisiones entre vender o usar
- da poder lateral sin destruir el estilo retro
- hace que ciertos monstruos y rutas de hunt ganen identidad

## Que conviene conservar al portarlo

- efectos faciles de entender
- stacks limitados
- slots claros por tipo de gema
- feedback visible al item o al jugador

## Riesgos

- sin topes, el balance se dispara
- si el fail chance es demasiado cruel, genera rechazo
- si las imbuements se pisan mal, aparecen combinaciones absurdas

## Portabilidad

Media. Muy buena feature, pero conviene reequilibrarla segun la economia del server destino.
