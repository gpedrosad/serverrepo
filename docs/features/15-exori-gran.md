# 15. Exori gran

## Que es

Spell custom para Knight pensado como upgrade claro de `exori`:

- mas area
- mas dano
- identidad visual mas fuerte

Se resolvio de forma data-driven, sin necesidad de rehacer el motor.

## Por que valio la pena

- da personalidad a la vocacion
- genera sensacion de progreso real
- es un ejemplo bueno de como agregar spells custom sin romper toda la base

## Que conviene conservar al portarlo

- que se sienta upgrade, no reemplazo obligatorio de todo
- identidad visual clara
- costo/requisito coherente con el premio

## Riesgos

- si el dano o area se pasan, rompe el balance del Knight
- si queda demasiado parecido a exori normal, no justifica existir

## Portabilidad

Alta. Es facil de portar si el otro server soporta spells por data o scripts.
